## Core Principles

</div>

<table>
<tr>
<td width="33%" align="center">

**100% Local Processing**

Every AI feature runs on your device. No data sent to remote servers. No internet required for core features.

</td>
<td width="33%" align="center">

**End-to-End Encryption**

Your data is encrypted before it ever leaves your device. Even cloud storage providers can't see your data.

</td>
<td width="33%" align="center">

**Consumer-Grade UX**

No command-line setup. No technical expertise required. Privacy tools should be as easy to use as any modern app.

</td>
</tr>
</table>

## Technology Stack

**Frontend:** React 19 · TypeScript · Vite · Tailwind CSS
**Editor:** TipTap (ProseMirror-based) · Custom Extensions
**Storage:** IndexedDB (local-first) · Dexie.js
**Encryption:** Web Crypto API · AES-256-GCM · PBKDF2
**Mobile:** Capacitor 7 · iOS · Android
**i18n:** react-i18next · RTL/CJK Support
